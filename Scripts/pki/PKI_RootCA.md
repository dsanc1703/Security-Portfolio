# Root CA Server Maintenance

### Purpose
Automates the high-risk manual steps required for multi-year maintenance of a ADCS Two-Tier PKI Root Certificate Authority Server. 
\
Ensures consistent file naming and directory placement across System and mounted drives.

### Specifics
* **Language:** PowerShell
* **Workflow:** Menu interface for Root CA renewal, CRL publishing, and Intermediate CA signing
