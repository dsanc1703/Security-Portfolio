# Intermediate CA Server Maintenance

### Purpose
Automates the high-risk manual steps required for multi-year maintenance of a ADCS Two-Tier PKI Intermediate Certificate Authority Server. 
\
Ensures consistent file naming and directory placement across System and Webserver (IIS) enrollment points.

### Specifics
* **Language:** PowerShell
* **Workflow:** Menu interface for Root CA imports, CSR generation, and CRL distribution.
