# Certificate Specification Scanner

### Purpose
Audits list of active HTTPS endpoints to identify the cryptographic strength of their certificate. 

### Specifics
* **Language:** PowerShell
* **Connection Method:** SslStream
* **Audit Scope:** Key Bit Length and Signature Algorithm
* **Output:** CSV file of cryptographic strength categorization
