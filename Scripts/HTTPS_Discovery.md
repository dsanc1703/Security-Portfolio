# HTTPS Discovery Scanner

### Purpose
Audits a list of ping-responsive assets (IP Addresses) to identify any active web interfaces.

### Specifics
* **Language:** PowerShell
* **Connection Method:** TCP via Test-NetConnection
* **Target Ports:** 80 and 443
* **Output:** CSV file of device categorization
